# Analysis-of-real-estate-market-in-SPB-and-Len-oblast


## 📊  Аналитический проект: Анализ рынка недвижимости Санкт-Петербурга и Ленинградской области

Это учебный проект Яндекс Практикума

**ЦЕЛЬ ПРОЕКТА**: анализ объявлений о продаже жилой недвижимости в Санкт-Петербурге и Ленинградской области, чтобы найти самые перспективные сегменты недвижимости.

---
### 📁СТРУКТУРА 
* **Исследовательский анализ данных**
* **Решение ad-hoc задач**
* **Выводы по проекту**
* **Визуализация при помощи BI инстурмента (Data Lens)Для этого учебного проекта было необходжимо решить две ad-hoc задачи и доработать дэшборд**
---
### 🔍 Содержание анализа
####  Часть 1. Исследовательский анализ данных
* Анализ выбросов и их фильтрация по 99 и 1 процентилю
####  Часть 2. Ad hoc задачи
* Выявление перспективных направлений на рынке недвижимости и анализ влияния характеристик квартиры на скорость её продажи
* Анализ сезонности рынка
* Исследование региональных особенностей для Ленинградской области
* Аналитические заметки по каждому блоку исследования
####  Часть 3. Визуализация
* Доработка чарта и добавление фильтров и диаграмм по т.з.
